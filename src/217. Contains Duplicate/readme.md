Given an integer array _`nums`_, return _`true`_ if any value appears **at least twice** in the array, and return _`false`_ if every element is distinct.

**Example 1:**

```
Input: nums = [1,2,3,1]
Output: true
```

**Example 2:**

```
Input: nums = [1,2,3,4]
Output: false
```

**Example 3:**

```
Input: nums = [1,1,1,3,3,4,3,2,4,2]
Output: true
```

**Constraints:**

- [x] `1 <= nums.length <= 105`
- [x] `-109 <= nums[i] <= 109`
